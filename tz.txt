ТЗ: Pokémon Web App
1. Главная страница

Цель: Отобразить список всех покемонов (1000 штук) в виде текста.

Элементы:

Список покемонов (имена).

Кнопка или ссылка на детальную страницу каждого покемона.

Функционал:

При клике на покемона — переход на его детальную страницу.

Использовать пагинацию или виртуальный скролл для производительности (1000 покемонов).

2. Детальная страница покемона

Цель: Показать информацию о выбранном покемоне.

Элементы:

Имя, тип, скиллы и качество скиллов.

Кнопка “Добавить в коллекцию”.

Функционал:

При добавлении покемон сохраняется в локальное хранилище через Zustand + LocalStorage.

Можно посмотреть, если покемон уже есть в коллекции.

3. Коллекция

Цель: Собирать покемонов.

Элементы:

Список покемонов из коллекции.

Кнопка удаления из коллекции.

Функционал:

Хранение коллекции в Zustand + LocalStorage, чтобы сохранялась между сессиями.

Фильтры: по имени, по типу, по силе.

4. Арена

Цель: Сражения между покемонами.

Элементы:

Два селектора для выбора покемонов из коллекции.

Кнопка “Сразиться”.

Результат боя.

Функционал:

Победитель определяется по сумме качества скиллов.

Можно отображать “боевую анимацию” или просто текстовый результат.
